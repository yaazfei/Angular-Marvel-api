# AngularMarvel

Aplicação Front-End em Angular (11.0.3) que acessa API da Marvel (https://developer.marvel.com/)

# Inclui: 
- Layout Responsivo
- Busca de Personagens
- Listagem de Personagens
- Página de Detalhes


## Development server

Iniciar server dev
```sh
ng serve
```

Endereço:
`http://localhost:4200/`
